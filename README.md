# 🧮 Gelişmiş Hesap Makinesi (Temel & Bilimsel)

Bu proje, Python (Flask) arka planı ve modern HTML/CSS/JS arayüzü ile çalışan, hem temel matematiksel işlemleri hem de karmaşık bilimsel fonksiyonları destekleyen gelişmiş bir hesap makinesi uygulamasıdır.

## 🚀 Özellikler

- **Temel İşlemler:** Toplama, Çıkarma, Çarpma, Bölme ve Mod Alma (`%`).
- **Bilimsel Fonksiyonlar:** `sin`, `cos`, `tan`, `log` (10 tabanında), `ln` (e tabanında), `sqrt` (karekök) ve üslü işlemler (`^`).
- **Matematiksel Sabitler:** $\pi$ (Pi) ve $e$ sayıları için doğrudan buton desteği.
- **Güvenli Hesaplama:** Arka planda dinamik `eval` yapısı, izole ve güvenli bir ortamda çalıştırılır.
- **Anlık Tepki Veren UI:** Saf JavaScript tabanlı tuş takımı sayesinde sıfır gecikmeli ekran deneyimi.

## 📦 Kurulum ve Çalıştırma

Projeyi yerel bilgisayarınızda çalıştırmak için aşağıdaki adımları takip edebilirsiniz:

1. **Bağımlılıkları Yükleyin:**
   pip install -r requirements.txt
2. **Uygulamayı Başlatın:**
   python ui.py
   Uygulama başladığında tarayıcınız otomatik olarak http://127.0.0.1:5000 adresinde açılacaktır.
3. **Kullanılan Teknolojiler:** 
   Backend: Python, Flask
   Frontend: HTML5, CSS3 (Koyu Tema), JavaScript (ES6)
